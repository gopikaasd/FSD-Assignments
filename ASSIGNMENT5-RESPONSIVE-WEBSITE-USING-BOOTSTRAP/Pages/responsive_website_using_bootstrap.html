Fixed Asset

A fixed asset project is an investment initiative, such as building a new factory or purchasing heavy machinery, where the costs are tracked and accumulated within a project management system until the project is complete and the asset is ready for operational use. At that point, the project is finalized, and the accumulated costs are then transferred and "capitalized" to a newly created fixed asset in the company's fixed asset register for subsequent depreciation and financial reporting..
